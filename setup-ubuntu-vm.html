Oppsett av Ubuntu Server VM

1. Installer Ubuntu Server

  1. Monter Ubuntu Server ISO til VM-en
  2. Start VM-en og begynn installasjonen
  3. Velg språk og tastaturoppsett
  4. Velg "Installer Ubuntu Server"

2. Nettverkskonfigurasjon

  • Konfigurer nettverksgrensesnitt
  • Sett statisk IP-adresse (anbefalt)
  • Konfigurer DNS-innstillinger

Eksempel på nettverkskonfigurasjon:

IP-adresse: 10.12.pool.100
Nettmaske: 10.12.pool.0/24
Gateway: 10.12.pool.1
DNS: 10.12.pool.10, 10.10.1.30, 1.1.1.1, 1.0.0.1
