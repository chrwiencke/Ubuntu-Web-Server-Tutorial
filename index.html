Velkommen til Ubuntu Webserver Veiledning

Denne veiledningen vil guide deg gjennom prosessen med å sette opp en Ubuntu webserver ved hjelp av:

  • VMware vSphere for virtualisering
  • Ubuntu Server som operativsystem
  • Nginx som webserver

Følg navigasjonsmenyen til venstre for å gå gjennom hvert trinn i oppsettet.

Forutsetninger:

  • Tilgang til VMware vSphere-miljø
  • Ubuntu Server ISO-fil
  • Grunnleggende kunnskap om Linux-kommandoer
