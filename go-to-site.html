Tilgang til Din Webserver

Åpne Nettsiden Din

Du kan nå få tilgang til nettsiden din ved å bruke:

http://din-server-ip http://tutorial.skynet.ikt-fag.no

Feilsøking

  • Verifiser at Nginx kjører: systemctl status nginx
  • Sjekk brannmurinnstillinger: sudo ufw status
  • Se Nginx feillogger: sudo tail -f /var/log/nginx/error.log

Gratulerer! Du har nå satt opp din Ubuntu webserver med Nginx!
