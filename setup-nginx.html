Oppsett av Nginx Webserver

1. Installer Nginx

sudo apt install nginx -y

2. Konfigurer Brannmur

Tillat HTTP og HTTPS trafikk:

sudo ufw allow 'Nginx Full'

3. Verifiser Installasjon

Sjekk Nginx status:

sudo systemctl status nginx

4. Grunnleggende Nginx Konfigurasjon

Rediger standardkonfigurasjonen:

sudo nano /etc/nginx/sites-available/default

Eksempel på grunnleggende konfigurasjon:

server { listen 80; listen [::]:80; root /var/www/html; index index.html; server_name _; location / { try_files $uri $uri/ =404; } }

Dette setter opp en enkel filserver som:

  • Lytter på port 80
  • Serverer filer fra /var/www/html
  • Bruker index.html som standardside
  • Aksepterer alle hostnavn

Husk å omstarte Nginx etter konfigurasjonsendringer:

sudo systemctl restart nginx
